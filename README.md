📚 Trabalho Final - Biblioteca IMA

📖 Sobre o Projeto

Este projeto tem como objetivo aprimorar o sistema da Biblioteca IMA, proporcionando uma melhor 
experiência para os usuários e facilitando o acesso ao acervo disponível.
 A biblioteca desempenha um papel fundamental na preservação e disseminação do conhecimento, 
 oferecendo suporte acadêmico e cultural.

🏛️ Descrição do Projeto

A Biblioteca do Instituto Mário Alves (IMA) em Pelotas desempenha um papel importante na 
preservação da cultura e memória histórica da região. Fundado em 2001, o Instituto de 
Estudos Políticos Mário Alves abriga uma biblioteca popular, uma videoteca alternativa e 
um arquivo histórico com documentos que retratam a organização dos movimentos sociais locais.
Além de seu acervo, o IMA promove atividades culturais, seminários, palestras e exposições 
fotográficas, contribuindo para a formação cidadã e o fortalecimento da identidade comunitária


🔧 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- Next.js – Framework para React que melhora a performance e a experiência do usuário.
- GitHub – Plataforma de versionamento e colaboração no desenvolvimento do sistema.
- JavaScript – Linguagem principal para a implementação das funcionalidades interativas.
- Python – Utilizado para automação e manipulação de dados.
- Tailwind CSS – Framework de estilização para interfaces modernas e responsivas.
- MySQL – Banco de dados relacional para armazenamento e gerenciamento das informações.

🎯 Funcionalidades

   - Cadastro de usuários: Permitir que usuários criem uma conta básica para acessar os serviços da biblioteca.
   - Consulta ao acervo: Implementar um sistema de busca simples para verificar os livros disponíveis.
   - Empréstimos e devoluções: Criar um registro básico de retiradas de livros, com data de empréstimo e 
   devolução.
   - Renovação de empréstimos: Disponibilizar a opção de renovar um livro online sem necessidade 
   de ir até a biblioteca.
   - Reservas de livros: Permitir que usuários reservem livros, garantindo que eles 
   estejam disponíveis na retirada.
   - Notificações básicas: Enviar lembretes sobre prazos de devolução e status de reserva.
